# Новогодний ЗАБЕГ 2026

Этот проект представляет собой лендинг для новогоднего забега.

## Разработка

1. Установите зависимости:
   ```bash
   npm install
   ```

2. Запустите локальный сервер:
   ```bash
   npm run dev
   ```

## Сборка для продакшена

Чтобы собрать проект для выкладки на VPS:

1. Выполните команду сборки:
   ```bash
   npm run build
   ```

2. После успешного выполнения в папке `dist` появятся файлы, готовые к размещению на сервере.

## Деплой на VPS

1. **Клонируйте репозиторий** на ваш сервер:
   ```bash
   git clone <your-repo-url>
   cd NYRun2
   ```

2. **Установите зависимости и соберите проект**:
   ```bash
   npm install
   npm run build
   ```

3. **Настройте веб-сервер** (например, Nginx) для раздачи статических файлов из папки `dist`.

   Пример конфигурации Nginx:
   ```nginx
   server {
       listen 80;
       server_name your-domain.com;

       root /path/to/NYRun2/dist;
       index index.html;

       location / {
           try_files $uri $uri/ /index.html;
       }
   }
   ```

## Безопасность

- В репозитории нет секретных ключей или токенов.
- Файл `.gitignore` настроен так, чтобы исключить `node_modules` и другие системные файлы.
